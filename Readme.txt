Python implementation of the paper "Online Multi-Kernel Learning with Graph-Structured Feedback" presented in ICML 2020.
mkl_algorithms.py: Python implementation of multi-kernel learning methods including OMKR, OMKR-FA, Raker and OMKL-GF.

AirData.mat: Air Quality dataset
IstanbulEx.mat: Istanbul Exchange dataset
TwitterData.mat: Twitter dataset
TomData.mat: Tom's Hardware dataset
NavalData.txt: Naval Propulsion Plants dataset

Table_1.ipynb: produces the results in Table 1 of the paper.
Table_2.ipynb: produces the results in Table 2 of the paper.
Table_3.ipynb: produces the results in Table 3 of the paper.
Table_4.ipynb: produces the results in Table 4 of the paper.
Table_5_Fig_1 : produces the results in Table 5 and Figure 1 of the paper.
